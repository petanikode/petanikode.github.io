Proyek Aplikasi Matuku

Ardianta Pargo dalam pemrograman
31 January 2015 · 21:14

Manajemen Tugas Kuliah (Matuku), itulah nama aplikasi yang saya kembangkan di proyek sederhana ini. Proyek ini untuk memenuhi tugas kuliah pemrograman web. Dalam tugas ini, saya diminta untuk membuat sebuah aplikasi dengan PHP, dilengkapi dengan fitur CRUD (CREATE, READ, UPDATE, DELETE). Struktur tabel minimal terdapat tujuh kolom; Tabel tidak berelasi. Di dalam aplikasi harus ada unsur lojik atau operasi.

Analisis Kebutuhan

Aplikasi ini akan dipakai oleh Mahasiswa. Oleh karena itu, saya perlu menganalisis kebutuhan pemakai. Di bawah ini adalah daftar kebutuhan fungsional pemakai.

  1. Pemakai dapat menambah, mengubah, menghapus tugas;
  2. Pemakai dapat melihat dan mensortir tugas yang sudah selesai dan yang belum selesai;
  3. Pemakai harus login dulu, baru dapat mengubah, melihat, menambahkan, dan menghapus tugas;
  4. Pemakai dapat melihat detail tugas dan deadline tugas;

Rancangan sistem

Proyek kecil-kecilan ini bukanlah sistem yang besar. Proses atau aktivitas yang dilakukan di sistem ini adalah CRUD dan login saja. Oleh karena itu, perancangannya sangat sederhana. Cukup dengan rancangan basis data dan interface, sistem ini sudah dapat diimplementasikan dan berjalan sesuai kebutuhan.

Rancangan basis data

Tabel yang diperlukan di proyek ini ada dua, yaitu tabel pemakai dan tabel tugas. Tabel tidak berelasi, jadi sangat mudah membuat rancangannya.

Rancangan interface

Ada beberapa halaman yang dibutuhkan untuk tampilan interface: halaman login, home, form input, form edit, dan halaman detail.

Penutup

Demikianlah tulisan perihal proyek pengembangan aplikasi Manajemen Tugas Kuliah ini. Masih banyak kekurangan dalam aplikasi ini, diantaranya:

  1. Pageination, untuk menangani tampilan data yang banyak.
  2. Ajax, Agar halaman tidak perlu di-reload.
  3. Setting, untuk mengubah password dan data user.
  4. Lupa Password, Fitur untuk reset password.

Harapan kedepan, dapat melengkapi segala kekurangan yang ada di aplikasi ini. Semoga saja dapat diwujudkan dan mudah-mudahan tulisan ini bermanfaat.
