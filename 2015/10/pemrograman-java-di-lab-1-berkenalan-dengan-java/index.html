Pemrograman java di lab (1): Berkenalan dengan java

Ardianta Pargo dalam pemrograman
7 October 2015 · 21:50


Hari ini adalah hari pertama praktikum pemrograman java dengan mahasiswa baru. Saya dari petani kode dipercaya untuk menjadi teaching assistant pada praktikum ini. Suka duka menjadi teaching assistant, kadang-kadang menjelma menjadi debugger yang bisa menemukan berbagai macam error.

Bagaimana pemrograman java itu?

Pemrograman java sebenarnya sangat sederhana. Menulis kode, kompilasi, lalu eksekusi atau jalankan program. Menulis kode dapat menggunakan teks editor. Namun, agar lebih mudah, kita akan menggunakan IDE Netbeans. Kemudian kompilasinya menggunakan JDK dan menjalankannya di JRE. Kurang lebih gambarannya seperti ini:

Membuat proyek baru di Netbeans

  1. Arahkan ke menu File kemudian pilih New Project. Pada jendela yang muncul, pilih Java Application kemudian klik next.
  2. Isi data proyek, cukup mengganti project name saja (Lab-Java) dan ‘Create Main Class’ tidak perlu dicentang. Setelah itu klik finish.

Membuat Paket

Kita akan menyimpan tiap kode java dalam paket agar tertata rapi dan mudah ditemukan. Nama paket akan selalu berubah-ubah setiap kali pertemuan. Pada pertemuan pertama ini nama paketnya adalah ‘pertemuan1’.

Pembuatan paket dapat dilakukan dengan mengklik kanan pada Source Package. Kemudian pilih new > Java Package.
Pada jendela yang muncul, isilah nama paketnya dengan ‘pertemuan1’. Setelah itu, klik Finish.

Prograrm Pertama

Program java dibuat dalam sebuah kelas (class). Jadi, kita perlu membuat kelas baru untuk membuat program. Buatlah sebuah kelas baru (Java Class) pada paket pertemuan1.

Pada jendela yang muncul, isilah nama kelasnya dengan ‘Perkenalan’. Setelah itu klik Finish.

Maka akan tercipta sebuah kelas baru bernama ‘Perkenalan’.

Teks yang berwarna abu-abu adalah sebuah komentar. Teks tersebut tidak akan berpengaruh pada program yang kita buat. Namun, hanya sebagai keterangan saja. Sementara, package pertemuan1 menandakan kalau kode java ini berada di dalam paket pertemuan1.

Kelas perkenalan dimulai dari tanda kurung “{” kemudian ditutup dengan tanda kurung “}”. Di dalam sanalah kita akan menulis programnya. Sebagai contoh, silahkan buat fungsi main() dalam kelas perkenalan, isinya berupa teks perkenalan. Fungsi main() adalah fungsi utama dalam sebuah program. Tampa fungsi ini, program tidak bisa dijalankan.

Fungsi System.out.println(“…”); adalah untuk menampilkan teks ke layar. Kita akan banyak memakai fungsi ini untuk menampilkan output. Kode di atas, akan menghasilkan output seperti ini:
