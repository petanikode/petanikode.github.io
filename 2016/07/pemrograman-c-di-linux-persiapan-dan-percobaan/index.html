Pemrograman C di Linux: Persiapan dan Percobaan

Ardianta Pargo dalam pemrograman
21 July 2016 · 12:11

Sebenarnya sudah lama sekali artikel ini tersimpan dalam tumpukan draf di blog ini. Tidak terbit-terbit sampai menjadi fosil. Oleh karena itu, dari pada disimpan terus, lebih baik saya terbitkan saja apa adanya.

Bahasa pemrograman C memang bahasa yang cukup tua dan tidak begitu populer saat ini. Tetapi tidak ada salahnya kita pelajari, karena bahasa yang populer saat ini berasal dari bahasa C.

Teknologi yang pupuler saat ini menurut survey stackoverflow.

Persiapan Pemrograman C di Linux

Ada dua peralatan yang harus dipersiapkan untuk memrogram C di Linux, diantaranya:

1. Teks Editor

Anda bebas menggunakan teks editor apa pun. Teks editor digunakan untuk menulis kode program C. Rekomendasi dari saya, bisa pakai Gedit, Geany, Mousepad, Kate, Atom, dsb.

2. Kompilator GCC

Kompilator fungsinya untuk mengubah kode bahasa C ke dalam program yang siap dieksekusi. Silahkan instal kompilator C di Linux dengan perintah:
sudo apt install  gcc
Atau, kalau mau yang lebih lengkap dengan build tools-nya ketik:
sudo apt install build-essential

Percobaan Pemrograman C di Linux

Percobaan kita lakukan dengan membuat sebuah program sederhana. Program yang sangat populer untuk mencoba-coba, namanya Hello World!. Silahkan buat kode di bawah ini dengan teks editor, kemudian simpan dengan nama hello_world.c.
#include <stdio.h>

int main(){
    printf(“Hello World!”);
    return 0;
}
Simpan di tempat yang sudah anda tentukan. Setelah itu buka terminal dan arahkan ke direktori tempat anda menyimpan kode tersebut.

Cara Kompilasi

Berikut ini perintah untuk mengompilasi kode program bahasa C di Linux:
gcc kode_program.c -o output_hasil
kode_program.c adalah nama file kodenya, sedangkan output_hasil adalah nama file hasilnya. Nanti hasilnya akan berupa file executable.

Cara Eksekusi Program

Berikut ini perintah untuk mengeksekusi atau menjalankan program di Linux:
./nama_program
Pastikan anda berada dalam direktori tempat program tersebut berada dan diberikan hak aksess untuk mengeksekusi.

Tampilannya ke samping karena lupa dikasih Enter.

Kesimpulan

Pemrograman C di Linux membutuhkan dua alat, yaitu teks editor dan kompilator. Kompilasi kode program C dilakukan melalui terminal atau bisa juga melalui IDE.
