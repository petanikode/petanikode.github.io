Belajar HTML: Cara Membuat Link untuk Menghubungkan Halaman

#HTML5 #HTML


Link adalah elemen html yang berfungsi menghubungkan suatu tempat ke tempat yang lain. Link ibarat pintu kemana saja. Setiap kali diklik, kita akan dibawa menuju ke suatu tempat. Dalam pembuatan web, link adalah salah satu elemen yang sering digunakan.

Penulisan Link

Link dapat dibuat dengan tag <a>. Untuk dapat menuju ke tempat lain, tag ini perlu ditambahkan atribut href. Dalam atribut inilah kita menulisakan alamat yang dituju. Di bawah ini adalah contoh link menuju ke google.
<a href=“https://google.com">Link ini menuju ke Google</a>
Isi atribut href dapat berupa URL, nama domain, id element HTML, nama berkas dan lain-lain. Selain menuju ke suatu tempat, link juga dapat digunakan untuk memanggil aplikasi tertentu. Misalkan memanggil aplikasi mail client untuk menulis surat.

Atribut-Atribut Link

Seain atribut href terdapat juga atribut yang lain seperti target, title, rel, dan lain-lain. Dalam tulisan ini saya akan membahas atribut yang sering digunkan di situs web.

Atribut target

Atribut ini berfungsi untuk memberi tahu link, target membukanya di mana. Misalkan targetnya adalah di jendela baru atau tab baru, maka atribut target harus diisi dengan blank atau _blank.
<a href=‘http://ardianta.tumblr.com' target=‘blank’>Balog tumblr</a>
Selain itu atribut ini dapat juga di isi dengan _parent, _self, _top dan nama sebuah frame. Contoh:
<a href=“http://petanikode.blogspot.com" target=“tampil”>Sebuah Link</a>
<iframe name=“tampil”>
</iframe>
Hasilnya akan dibuka di dalam elemen frame yang memiliki atribut name=“tampil”

Atribut title

Atribut ini berfungsi untuk membuat tooltips. Tooltips adalah informasi yang tampil saat link di lewati (disentuh) pointer.
<a href=“mailto:ardianta_pargo@yahoo.co.id” title=“kirim email ke Ardianta Pargo”>Hubungi Saya</a>
Link di atas akan memanggil aplikasi mail client, karena saya mengisi nilai href dengan mailto:ardianta_pargo@yahoo.co.id.
