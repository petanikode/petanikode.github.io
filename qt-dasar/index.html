Pemrograman GUI dengan Qt: Persiapan dan Pengenalan

#C++ #Qt

Program Hello World seringkali dibuat saat pertama kali mempelajari suatu bahasa pemrograman. Program ini begitu legendaris dan populer di dunia pemrograman. Namun, dalam tulisan ini, saya akan mengajak pembaca membuat program kenalan di Qt. Supaya tidak mainstream.

Buat sebuah proyek baru bernama kenalan. Silahkan menuju ke menu File->New File or Project... atau tekan tombol Ctrl+N. Kemudian pilih Application->Qt Widget Application, lalu klik tombol Choose....

Selanjutnya, menentukan nama proyek dan lokasi penyimpanannya. Berikan nama kenalan, kemudian klik Next.

Menentukan kit, biarkan saja apa adanya kemudian klik Next.

Selanjutnya, mengisi informasi kelas. Buatlah kelas dengan JendelaUtama.

Selanjutnya akan tampil informasi rangkuman proyek yang dibuat. Klik Finish.

Proyek baru-pun tercipta. Silahkan buka berkas jendelautama.ui untuk mendesain tampilan aplikasi.

Masukkan objek QLabel ke jendela dengan menyeretnya (drag). Kemudian ganti nilai properti font dan text di bagian objek property.

Silahkan mainkan imajinasi Anda. Saya sendiri merubah beberapa properti. Saya merubah properti teks QLabel yang pertama (label) menjadi Assalamualaikum dan memperbesar ukuran fontanya.

Sedangkan pada QLabel yang kedua (label_2), saya merubah properti teksnya dan wordWrap menjadi true (dicentang) supaya teks dapat tampil lebih dari satu baris.

Sekarang coba jalankan program dengan menglik tombol Run atau menekan tombol Ctrl+R.

O, ya, saat menjalankan program ini, saya mendapatkan galat üëé error: cannot find -lGL. Karena pustaka openGL belum diinstal di komputer saya. Silahkan baca solusinya di tulisan ini.
